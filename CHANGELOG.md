# Changelog

All notable changes to this extension will be documented in this file.

## 0.0.14

- Stable version used as base for the public GitHub repository.
- Git submenu positioned in a natural group in the Explorer context menu (`group: "3_git"`).
- Keeps:
  - **Add** using Git CLI (`git add`) with multi-select support.
  - **Rollback…** using Git CLI (`git restore`/`git checkout`) with multi-select support.
  - Commands: Commit, Add, Add All, Show Diff, Rollback, New Branch, Push, Pull.
  - Custom GoLand-style icon.

## 0.0.13

- README updated to be simpler and more aligned with Marketplace style.

## 0.0.12

- Adjusted submenu position to a more natural location in the Explorer (`group: "3_git"`).

## 0.0.11

- Added a custom GoLand-style icon to the extension.

## 0.0.10

- Updated **Rollback…** to use Git CLI (`restore`/`checkout`) with multi-select support.

## 0.0.9

- Updated **Add** to use Git CLI (`git add`) with multi-select support.

## 0.0.8
